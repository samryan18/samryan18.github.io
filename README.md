# samryan18.github.io

Personal website for Samuel Ryan — hosted on GitHub Pages.

## Structure

- `index.html` — Single-page site with inline CSS and JS
- `assets/images/` — Photography and portrait images

## Features

- Minimal, typography-focused design using EB Garamond
- Expandable sections (intro, links, photo galleries)
- Photo galleries from PCT 2021 thru-hike and other adventures
- Responsive layout with reduced-motion support

## Development

Static site — no build step. Open `index.html` in a browser or push to `main` to deploy via GitHub Pages.
