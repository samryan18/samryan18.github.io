Intro

Welcome. I built this personal site on a long flight a while back and generally keep the tradition by only updating it when I fly (infrequently).

I write some code. I hope to start writing more sentences too. Software can help solve optimization problems but doesn't help much with figuring out the right objective functions. The goal is to thrive. Let's make sure Moloch doesn't win.
